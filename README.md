# СочиГУ — концепция главной страницы (тестовое задание)

Адаптивная вёрстка главной страницы Сочинского государственного университета.

## Как открыть

1. Откройте файл `index.html` в браузере  
   **или**
2. Запустите локальный сервер из этой папки:

```bash
npx --yes serve .
```

Затем откройте адрес из терминала (обычно `http://localhost:3000`).

## Структура

```
sochi-sgu-homepage/
├── index.html
├── css/styles.css
├── js/main.js
├── assets/logo.svg
├── README.md
└── REPORT.md
```

## Проверено

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Ширины: 375px, 768px, 1200px+

## JavaScript

- Мобильное меню
- Плавный скролл по якорям
- Подсветка активного пункта меню
- Анимация появления блоков при скролле
- Кнопка «Наверх»
- Модальные карточки программ («Смотреть программу»)

Подробности концепции, палитры и UX-анализа — в `REPORT.md`.
